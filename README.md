#Android RSS Reader

## Build Requirements
    * JDK 1.6 or higher
    * Maven 3.0.4  or higher
    * Android SDK 14

## Build and Run app

### Build app
    * mvn clean install

### Start the emulator using the Android Maven Plugin
    * mvn android:emulator-start

### Deploy the app to the emulator
    * mvn android:deploy

### Start the sample app
    * mvn android:run
